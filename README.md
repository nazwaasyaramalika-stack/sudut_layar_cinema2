# 🎬 The Cinema Edu - Sudut Layar Edukasi

Selamat datang di **The Cinema Edu**, sebuah platform web kurasi film yang menggabungkan estetika desain klasik dengan kemudahan akses informasi sinema. Proyek ini dirancang untuk memberikan pengalaman membaca dan menjelajah dunia film dengan tampilan yang elegan dan responsif.

## ✨ Fitur Utama
- **Desain Klasik & Elegan:** Menggunakan tipografi *Cormorant Garamond* dan *Bebas Neue* untuk nuansa sinematik yang kuat.
- **Navigasi Mulus:** Transisi antar halaman yang halus dengan efek *custom cursor*.
- **Koleksi Terkurasi:** Menampilkan film-film pilihan seperti *Oppenheimer*, *Past Lives*, dan *The Zone of Interest*.
- **Login Modal:** Antarmuka login yang interaktif dengan efek animasi saat terjadi kesalahan.
- **Responsif:** Tampilan yang optimal baik di desktop maupun perangkat seluler.

## 🛠️ Teknologi yang Digunakan
- **HTML5:** Struktur konten web.
- **CSS3:** Styling kustom menggunakan variabel CSS, Flexbox, dan Grid.
- **JavaScript:** Logika fungsional untuk modal login, kursor kustom, dan efek animasi.
- **Google Fonts:** Integrasi font eksternal untuk tipografi premium.

## 🚀 Cara Menjalankan Proyek Secara Lokal
Jika Anda ingin mencoba web ini di komputer Anda sendiri:
1. Klik tombol **Code** (hijau) di bagian atas repositori ini.
2. Pilih **Download ZIP**.
3. Ekstrak file yang sudah didownload.
4. Klik dua kali pada file `index.html`.

## ✍️ Catatan Pengembang
Proyek ini dibuat sebagai bagian dari eksplorasi desain web yang mengutamakan kenyamanan membaca (*readability*) dan estetika visual yang terinspirasi dari poster-poster film klasik.
